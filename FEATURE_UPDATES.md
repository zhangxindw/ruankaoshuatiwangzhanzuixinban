### 2. 错题本功能增强

#### 功能说明
- ✅ 支持按选择的章节刷错题
- ✅ 支持一键刷全部章节的错题
- ✅ 章节选择对话框，显示每个章节的题目数量
- ✅ 批量选择功能（全选/取消全选）

#### 使用方式
1. 进入错题本页面
2. 点击"刷错题"按钮
3. 在弹出的对话框中选择要练习的章节
   - 可以勾选"全选所有章节"一次性选择所有章节
   - 可以单独选择特定章节
   - 每个章节显示题目数量
4. 点击"开始练习"
5. 系统会从所选章节的错题中随机抽取题目

#### 技术实现
- **后端**: 修改 `/api/wrong-questions/practice` 接口
  - 新增 `chapter_ids` 参数，支持按章节过滤错题
  - 支持多选章节
  - 自动打乱题目顺序
- **前端**: 
  - 新增章节选择对话框组件
  - 使用 checkbox-group 实现多选
  - 实时显示已选择章节数量
  - 调用后端 API 获取指定章节的错题

#### 文件修改
- `backend/app.py` - 修改 practice_wrong_questions 函数
- `frontend/src/views/WrongQuestionsView.vue` - 添加章节选择功能
- `frontend/src/store/quiz.js` - 无需修改（已支持）

## API 变更说明

### 1. GET /api/chapters

**响应变化：**
```json
{
  "status": "ok",
  "data": [
    {
      "id": 1,
      "name": "第一章",
      "order": 1,
      "parent_id": null,
      "question_count": 150  // 新增字段
    }
  ]
}
```

### 2. POST /api/wrong-questions/practice

**请求参数变化：**
```json
{
  "user_id": "default_user",
  "shuffle": true,
  "chapter_ids": [1, 2, 3]  // 新增参数，可选
}
```

**功能说明：**
- 不传 `chapter_ids` 或传空数组：获取所有错题
- 传入章节 ID 数组：只获取指定章节的错题
